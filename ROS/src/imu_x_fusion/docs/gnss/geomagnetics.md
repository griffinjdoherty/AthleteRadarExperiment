## Overview

随着地理位置的不同，通常地磁场的强度是 0.4-0.6 Gauss；而一个普通的手机喇叭当相距2厘米时仍会有大约4高斯的磁场，一个手机马达在相距2厘米时会有大约6高斯的磁场。
